Задачей агента поиска текущего состояния заданного объекта является поиск текущего состояния заданного объекта. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска текущего состояния заданного объекта. Единственным аргументом запроса является сущность.
