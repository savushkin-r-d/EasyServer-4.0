Задачей агента поиска временных границ указанной временной сущности является поиск временных границ указанной временной сущности. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска временных границ указанной временной сущности. Первым и единственным аргументом запроса является временная сущность.
